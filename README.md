# .github
Organization-wide templates, workflows, and automation


```
.github/
├── workflows/
│   ├── docs-deploy.yml (auto-deploy docs)
│   ├── docs-link-check.yml (verify links)
│   ├── docs-spellcheck.yml (optional)
│   └── auto-merge-dependabot.yml
├── ISSUE_TEMPLATE/
│   ├── bug_report.yml
│   ├── feature_request.yml
│   └── documentation_update.yml
├── PULL_REQUEST_TEMPLATE.md
├── CONTRIBUTING.md
├── CODE_OF_CONDUCT.md
└── SECURITY.md
```
